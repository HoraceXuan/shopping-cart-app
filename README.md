# 🛒 Shopping Cart Application

## 📌 Overview
This is a full-stack e-commerce shopping cart application built using React, Node.js, and MySQL.

Users can:
- View products
- Add items to cart
- Update quantity
- Remove items
- Track monthly purchases

---

## ⚙️ Tech Stack

- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: MySQL

---

## 🚀 Features

### Product List
- Display all products from database

### Shopping Cart
- Add to cart
- Increase / decrease quantity
- Remove items
- Auto calculate total

### Monthly Bill
- Record purchase history
- Show monthly total spending
- Display purchase timestamps

---

## 📂 Project Structure

- Frontend: `src/`, `public/`
- Backend: `server.js`, `db.js`
- Database: `shopping_cart_db.sql`
